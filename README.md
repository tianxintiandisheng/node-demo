## 调试

**使用 --inspect 参数运行你的 Node.js 脚本：**
```shell
node --inspect-brk your-script.js
```

**在 Chrome 地址栏中输入以下地址并回车：**
```shell
chrome://inspect
```
**点击 "inspect" 按钮**
点击你的脚本对应的 "inspect" 按钮，这将打开一个新的 Chrome DevTools 窗口。

**设置断点并调试**
在 DevTools 中，你可以设置断点、查看变量值、单步调试等，就像调试前端代码一样。
